Fixed columns in v-data-table demo

Note: fixed columns and grouping tables below are added only in my vuetify folk, which is published on npm. Also checkout more detailed grouping table demo

0. Regular Table(without fixed columns)

1. Regular Table with fixed columns

Fixed columns applied onto a regular table. grouping and checkboxes are not involved.

2. Selection(checkbox) Table with fixed columns

Fixed column applied onto a regular table with a checkbox column. No select all header. It expects an empty header with width to be specified which is used to render a regular empty header for checkbox column: { text: '', fixed: true, width: '100px', sortable: false }. Note: sortable:false is required

3. Selection(checkbox) Table with fixed columns and select all

Fixed column applied onto a regular table with a checkbox column. Has select all header. Similar to #2, it also expects an empty header with width to be specified which is used to render a 'select all' header for checkbox column: { text: '', fixed: true, width: '100px'}. Note: sortable:false is no longer required
